סיכום אינטואיטיבי למבחן בביולוגיה

הדף הזה עכשיו בנוי כמו מפת דרך אמיתית של הקורס: מהכימיה, דרך DNA ו-RNA, עד בניית חלבון, חלוקת תא, גנטיקה, בקרה ושיטות מעבדה. המטרה: לראות איפה כל שאלה יושבת במפה, ואז לענות מהר.

עברית מימין לשמאל. רצפים מדעיים נשארים משמאל לימין: DNA → RNA → Protein, 5' → 3', חלבון N → C.

המסלול המרכזי: מגן לחלבון

DNA gene -> transcription by RNA polymerase -> pre-mRNA -> 5' cap + splicing + poly-A -> mature mRNA -> ribosome reads codons from AUG -> tRNA anticodons bring amino acids -> amino-acid chain -> folding + modifications -> functional protein

איך לפתור שאלה במבחן

  1. לזהות תחנה במפה: כימיה, חלבון, אנזים, אנרגיה, DNA/RNA, חלוקת תא, גנטיקה, בקרה או שיטה.
  2. לסמן כיוון: 5'→3', N→C, למעלה/למטה בג'ל, או upstream/downstream במסלול.
  3. להפעיל כלל אחד בלבד.
  4. לבדוק היגיון: מטען, גודל, כיוון, יחס צאצאים, ATP או סימן של ΔG.
