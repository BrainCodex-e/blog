הדף הזה מכסה את המצגות והתרגולים: מושגים, ניסויים, תמונות, ג'לים, גרפים ושאלות מעבדה. בעברית נוח לכתוב תהליכים עם `←`; רצפים מדעיים נשארים משמאל לימין, למשל `DNA → RNA → Protein` ו-`5' → 3'`.

## ציר הקורס

```text
אותות ← הנדסה/ריצוף ← גנטיקה ← DNA ← אנרגיה ← אנזימים ← חלבונים ← כימיה
```

## כימיה, תמיסות, pH ובופרים

| נושא | מה צריך לזכור |
|---|---|
| אטום | פרוטונים קובעים יסוד; אלקטרונים קובעים תגובתיות; איזוטופים שונים במספר נייטרונים. |
| קשרים | קוולנטי חזק; יוני בין מטענים; קשר מימן עם H הקשור ל-O/N/F; ואן דר ולס חלש אך מצטבר. |
| אלקטרושליליות | הפרש גדול יוצר קוטביות; מולקולות פולריות אוהבות מים. |
| הידרופובי/הידרופילי | הידרופובי נדחה ממים; הידרופילי יוצר אינטראקציות עם מים. |
| מולריות | `1 mol = 6.022 × 10^23`, ו-`M = mol/liter`. |

חומצה מוסרת `H+`, בסיס קולט פרוטון או משחרר `OH-`.

```text
pH = pKa + log([A-]/[HA])
pH < pKa → protonated
pH > pKa → deprotonated
```

בופר הוא זוג חומצה חלשה ובסיס מצומד שמתנגדים לשינוי pH. בטיטרציה מחפשים את נקודת השקילות ואת אזור הבופר.

## חלבונים, ממברנות ושומנים

חלבון הוא שרשרת חומצות אמינו שמתקפלת. המבנה הוא התפקוד.

| רמה | פירוש |
|---|---|
| ראשוני | רצף חומצות אמינו, מ-N-terminus ל-C-terminus. |
| שניוני | α-helix ו-β-sheet. |
| שלישוני | קיפול תלת-ממדי של חלבון אחד. |
| רבעוני | כמה תתי יחידות, למשל המוגלובין. |

קבוצת `R` קובעת אם חומצה אמינית הידרופובית, הידרופילית, חיובית או שלילית. מוטציה באתר פעיל, באתר קישור, או באזור שמייצב קיפול משמעותית במיוחד.

| שומנים וממברנה | מה לזכור |
|---|---|
| טריגליצריד | גליצרול + 3 חומצות שומן, מאגר אנרגיה. |
| פוספוליפיד | ראש הידרופילי וזנבות הידרופוביים, בונה ממברנה. |
| רווי | אין קשר כפול, ישר יותר ונארז בצפיפות. |
| לא רווי | יש קשר כפול, יוצר כיפוף ומעלה נוזליות. |
| הובלה | דיפוזיה פשוטה עם מפל; דיפוזיה מזורזת עם חלבון; הובלה פעילה נגד מפל ודורשת אנרגיה. |

## אנזימים וקינטיקה

אנזים מוריד אנרגיית שפעול ולכן מזרז תגובה. הוא לא משנה את `ΔG`, לא משנה את שיווי המשקל, ולא נצרך בתגובה.

```text
E + S ⇌ ES ⇌ E + P
ΔG = ΔG° + RT ln([products]/[reactants])
```

| מושג | פירוש |
|---|---|
| `Vmax` | מהירות מקסימלית כשהאנזים רווי. |
| `Km` | ריכוז סובסטרט שבו `V0 = 1/2 Vmax`. |
| `Km` נמוך | אנזים עובד טוב גם בריכוז סובסטרט נמוך. |
| עיכוב תחרותי | `Km` עולה, `Vmax` כמעט לא משתנה. |
| עיכוב לא תחרותי | `Vmax` יורד. |
| אלוסטריה | קישור באתר אחר משנה מבנה ופעילות. |

דוגמת TIM: שינוי קטן באתר הפעיל או בלולאה שמייצבת את הסובסטרט יכול להוריד מהירות פי מאות או אלפים.

## אנרגיה, ATP ונשימה תאית

| שלב | מיקום | תוצרים |
|---|---|---|
| גליקוליזה | ציטופלזמה | 2 פירובט, 2 ATP נטו, 2 NADH. |
| פירובט → Acetyl-CoA | מיטוכונדריה | CO2 ו-NADH. |
| קרבס | מיטוכונדריה | NADH, FADH2, GTP, CO2. |
| זרחון חמצוני | מיטוכונדריה | רוב ה-ATP; חמצן הוא מקבל האלקטרונים הסופי. |
| תסיסה | בלי חמצן | רק 2 ATP מגליקוליזה; פירובט הופך ללקטט או אתנול ו-CO2. |

מגלוקוז אחד בנוכחות חמצן מתקבלים בערך `30-32 ATP`. חומצות שומן עוברות β-oxidation והופכות ל-Acetyl-CoA.

## גנטיקה, מיוזה ורקומבינציה

| כלי | איך משתמשים בו |
|---|---|
| ריבוע פנט | כותבים גמטות של כל הורה ומחשבים יחס גנוטיפי ופנוטיפי. |
| מבחן הכלאה | דומיננטי לא ידוע × רצסיבי. אם יש צאצאים רצסיביים, הדומיננטי היה הטרוזיגוט. |
| הסתברות | אירועים בלתי תלויים מכפילים; אפשרויות חלופיות מחברים. |
| אילן יוחסין | רצסיבי יכול לדלג דורות; דומיננטי נוטה להופיע בכל דור; X-linked נפוץ בזכרים. |

| תהליך | תוצאה |
|---|---|
| מיטוזה | 2 תאי בת זהים ודיפלואידיים. |
| מיוזה | 4 תאי מין שונים והפלואידיים. |
| רקומבינציה | החלפת מקטעים בין כרומוזומים הומולוגיים. |
| תאחיזה | גנים קרובים על אותו כרומוזום נוטים לעבור יחד. |

```text
1% recombination = 1 map unit = 1 cM
```

## DNA, RNA, שכפול, שעתוק, תרגום ומוטציות

```text
DNA → RNA → Protein
```

| נושא | נקודות חובה |
|---|---|
| DNA | A-T, C-G; בין C-G שלושה קשרי מימן ולכן GC יציב יותר. |
| RNA | U במקום T, סוכר ריבוז, לרוב חד-גדילי. |
| שעתוק | RNA polymerase יוצר RNA משלים לגדיל התבנית; RNA נבנה `5' → 3'`. |
| עיבוד באאוקריוטים | `5' cap`, זנב `poly-A`, שחבור אינטרונים וחיבור אקסונים. |
| תרגום | ריבוזום קורא קודונים; `AUG` התחלה; `UAA/UAG/UGA` עצירה. |
| מסגרת קריאה | מתחילים מ-AUG ומחלקים לשלשות. |

שכפול DNA הוא שמרני למחצה. DNA polymerase חייב פריימר ובונה רק `5' → 3'`. leading strand נבנה ברצף; lagging strand נבנה במקטעי Okazaki. Helicase פותח, primase שם פריימר, polymerase מאריך, RNase מסיר פריימרים, ligase מחבר, topoisomerase משחרר מתח.

| מוטציה | השפעה |
|---|---|
| Silent | אותה חומצה אמינית. |
| Missense conservative | חומצה אמינית דומה. |
| Missense non-conservative | חומצה אמינית שונה בתכונות. |
| Nonsense | קודון עצירה מוקדם. |
| Insertion/deletion לא כפולה של 3 | frameshift, לרוב חמור. |
| פרומוטר/שחבור/poly-A | פוגע בביטוי גם בלי לשנות את רצף החלבון. |

| ניסוי | מה עשו | מסקנה |
|---|---|---|
| Griffith | S מומתים בחום + R חיים הפכו ל-S חיים. | יש עקרון טרנספורמציה. |

## בקרה על ביטוי גנים, lac operon ומעבר אותות

כל התאים בגוף מכילים כמעט אותו DNA, אבל מבטאים גנים שונים. בקרה יכולה להיות ברמת כרומוזום, שעתוק, שחבור, יציבות mRNA, תרגום ועיבוד חלבון.

E. coli מייצר β-galactosidase ו-permease רק כשיש לקטוז ואין גלוקוז.

| מצב | תוצאה |
|---|---|
| אין לקטוז | רפרסור קשור לאופרטור, אין שעתוק. |
| יש לקטוז ויש גלוקוז | רפרסור משתחרר אבל CAP לא פעיל, ביטוי נמוך. |
| יש לקטוז ואין גלוקוז | רפרסור משוחרר ו-CAP/cAMP פעיל, ביטוי גבוה. |

mRNA מופיע לפני חלבון ונעלם מהר יותר; חלבון יכול להישאר גם אחרי שהשעתוק ירד.

```text
Growth factor ← RTK dimerization ← SOS/GEF ← RAS-GTP ← RAF/MEK/ERK ← שינוי ביטוי גנים
```

RAS-GDP לא פעיל; RAS-GTP פעיל. GEF מחליף GDP ב-GTP; GAP מעודד פירוק GTP ל-GDP. מוטציות שמחזיקות RAS או רצפטור פעילים מדי עלולות לגרום לסרטן.

## הנדסה גנטית, ג'לים, ריצוף, PCR, ChIP-seq ו-CRISPR

| כלי | מה חייבים לדעת |
|---|---|
| פלסמיד | DNA מעגלי; צריך origin, סמן ברירה, אתר חיתוך, ולעיתים פרומוטר/reporter. |
| אנזימי רסטריקציה | חותכים רצפים ספציפיים; שני אנזימים נותנים הכנסת כיוון נכונה. |
| ליגאז | מחבר קצוות DNA בקשר קוולנטי. |
| cDNA | נוצר מ-mRNA; שימושי לביטוי גן אאוקריוטי בחיידקים כי אין אינטרונים. |
| ג'ל | DNA שלילי ורץ לפלוס; קטן רץ רחוק יותר; פס עבה = יותר DNA. |
| PCR | דנטורציה 95°C, איחוי פריימרים ~68°C, הארכה 72°C; אחרי n מחזורים בערך `2^n`. |
| סנגר | ddNTP חסר 3'OH עוצר הארכה; בג'ל קוראים מלמטה למעלה. |
| ChIP-seq | קיבוע חלבון-DNA ← שבירה ← נוגדן ← ניקוי DNA ← ספרייה ← ריצוף ← peaks. |
| Knock-out | הרס גן, loss of function. |
| Knock-in | הכנסת/תיקון רצף, gain/change of function. |
| Cre-LoxP | conditional knock-out ברקמה או בזמן מסוים. |
| CRISPR-Cas9 | guide RNA מכוון את Cas9; תיקון לא מדויק יוצר knock-out, תיקון עם תבנית מאפשר knock-in. |

ספרייה גנומית כוללת גם אינטרונים ואזורי בקרה. ספריית cDNA כוללת רק גנים שהתבטאו בתא ובזמן הדגימה.

## איך לקרוא תמונות, גרפים ושאלות ניסוי

| תמונה/שאלה | איך לפתור |
|---|---|
| מבנה כימי | כל פינה היא C, משלימים H עד 4 קשרים, מחפשים O/N/F ומטענים. |
| גרף pH/בופר | באזור שבו pH משתנה לאט יש בופר; ב-`pH=pKa` יש חצי HA וחצי A-. |
| גרף אנזים | בודקים רוויה, `Vmax`, `Km`, ומה השתנה עם מעכב. |
| ג'ל רסטריקציה | מחברים גדלי פסים; חיתוך אחד במקטע לינארי נותן שני פסים. |
| סנגר | קוראים פס תחתון ראשון, ואז עולים; שומרים על כיוון `5' → 3'`. |
| אילן יוחסין | בודקים דילוג דורות, מין החולים, והאם אב מעביר לבן. |
| מסלול אותות | מסמנים upstream/downstream; מוטציה פעילה תמיד downstream תיתן תגובה גם אם upstream חסום. |
| בקרה גנטית | mRNA מוקדם וקצר חיים; חלבון מאוחר ויציב יותר. |

תבנית ניסוי: מה השאלה, מה שינו, מה מדדו, מה הביקורת, ואיזו תוצאה מחזקת או מפריכה את ההשערה.
